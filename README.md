# 📘 CodeTrek

A personal journey to master **Data Structures, Algorithms, and Problem-Solving** 🚀.
This repo contains **clean implementations, detailed notes, and patterns** for common coding interview problems.

---

## 📂 Repository Structure

```
CodeTrek/
│── arrays/           # Array-based problems & solutions
│── strings/          # String manipulation problems
│── linkedlist/       # Linked list implementations & problems
│── stacks_queues/    # Stack & queue problems
│── recursion/        # Recursion & backtracking problems
│── dynamic_programming/ # DP solutions with notes
│── notes/            # @notes style explanations for quick revision
│── utils/            # Helper functions if needed
```

## 🧑‍💻 Contents

* ✅ **Problem Implementations** → Python solutions for LeetCode/DSA patterns.
* 📝 **@notes** → Handcrafted explanations with complexity + intuition.
* 🔄 **Patterns Covered**

  * Two Pointers

  * Graph & Tree basics (WIP)

## 🚀 How to Use

Clone the repo:

```bash
git clone https://github.com/NaveenBasavaraj/CodeTrek.git
cd CodeTrek
```

Run a file (example):

```bash
python arrays/two_sum.py
```
